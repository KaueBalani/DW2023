passo a passo GitHub:
-------------------------------------------------------------------------
1-Entrar no GitHub com email e senha.
2-Entrar no VSCode.
3-Fechar pasta (Se estiver aberto) -File-->Close Folder.
4-Ir no botão "Source Control".
5-Botão "Clone Repository".
6-Copiar url do repositorio do GitHub e colar no VSCode e teclar "Enter".
7-Clonar o diretório em C:/xampp/htdocs e clicar em "open".
8-Confiar no autor.
-------------------------------------------------------------------------
---------------------------Configurar terminal---------------------------
1-Ir no menu Terminal-->new terminal.
2-digitar os comandos:
git config --global user.email "meu email".
git config --global user.name "meu nickname".
3-Fechar Terminal.
-------------------------------------------------------------------------
---------------------------Enviar no GitHub------------------------------
1-Salvar as mudanças "CTRL + S".
2-Ir no "Source Control".
3-Em "Changes" clicar "+".
4-Colocar mensagem no commit.
5-Clicar na setinha do commit.
6-Clicar em "Commit & push".
7-Na primeira vez dar as autorizações - na segunda não será mais necessário.
8-Ver no GitHub se foi atualizado.
-----------------------------------FIM--------------------------------------
