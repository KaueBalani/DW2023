Linguagem de programação quântica:

Linguagem de computação quântica (QCL) é uma das primeiras linguagens de programação quântica. A característica mais importante do QCL

é o suporte para operadores e funções definidos pelo usuário. A sua sintaxe é semelhante à sintaxe da linguagem de programação C e seus

tipos de dados clássicos são semelhantes aos tipos de dados primitivos em C. Pode-se combinar código clássico e código quântico no mesmo programa.
